# Blog-El-Camino-de-las-Big-Tech
### description
Blog about Story, Success and Failed's Big Tech in the Technology Sector

### features
Use CCBV (Class-Based-View) for manage Blog's in user interface
- Create a New Blog in the Site with: Name, Image (Optional), Description / Content and Date created
- Modify a Content or Name Blog
- Delete Blog by Site
